Self-Host Guide

FAQ

Is this a production email server?+

No. Inboxed is a development and testing tool. It catches emails — it never delivers them. It's a readable /dev/null.

What do I need to self-host?+

Any machine with Docker. A $6/month Hetzner VPS is more than enough. bin/setup && docker compose up -d and you're done.

What is MCP and why does it matter?+

MCP lets AI agents call tools directly. Your agent calls extract_code("test@mail.inboxed.dev") and gets the OTP. Works with Claude Code, Cursor, n8n.

Can I use Inboxed in CI/CD?+

Yes. Use the REST API. POST /api/v1/emails/wait long-polls until the email arrives — no hardcoded sleeps.

Is my data safe on the cloud trial?+

Cloud data has a 1-hour TTL and is auto-purged. No HTML rendering in cloud mode. Self-host for full control — no telemetry, no phone-home.
